gennai.docker
=========

gennai cluster.

## require

* [docker-compose](https://docs.docker.com/compose/)

## How

    $ git clone https://github.com/siniida/gennai.docker
    $ cd gennai.docker
    $ ./rebuild.sh
    $ docker-compose up
